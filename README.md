# Vis Board

### Deep dive into anything with Vis Board!

#### Preview:

![preview](shot.png)

### Features:

- Distill knowledge from messages
- Text/voice support
- Fully configurable
- Add/delete projects

### How It Works:

- Everything is saved in the user's browser
- Users must provide their OpenAI compatible Api in **Settings** to enable functionality
